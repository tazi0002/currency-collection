# Currency Collection

Simple inventory website for a coin and bank note collection.

## Features

- Filter by coin or bank note
- Filter by year
- Filter by collection set
- Filter by country
- Filter by Canadian value / denomination
- Search inventory

## Inventory updates

Inventory items are stored in `js/inventory.js`.
Future coin images should go in `images/coins/`.
Future bank note images should go in `images/banknotes/`.
